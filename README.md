## Celestia

숲으로 둘러 쌓인 마법 학교를 소재로 한 RPG 게임입니다.

마법 학교에 입학한 주인공이 주변 숲의 이상 현상 조사를 과제로 수행하면서 이상 현상을 일으킨 주범을 찾아내는 것이 주 목적입니다.
